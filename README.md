### 1: Install and Run Server

  - prepare nodejs env
  - `npm install -g gulp`
  - `npm install gulp -save-dev`
  - `npm install gulp-webserver -save-dev`
  - add `gulpfile.js`, `app/index.html`
  - run server: `$ gulp`
